# LiteLizard icon concepts

Source reference: user-provided raster concept (June 2026).

These files are design explorations only. They do not replace the application
icon in `apps/desktop/build/`.

## Faithful trace

- `faithful-vector-trace.svg`: Transparent-background vector trace focused only
  on the lizard and fountain pen from the supplied raster artwork. Its parts
  remain separated for later editing.

## Concepts

- `concept-a-faithful.svg`: Closest to the reference image. Keeps the eye,
  mouth, organic lizard silhouette, gold arc, and detailed pen nib.
- `concept-b-refined.svg`: Cleans up the silhouette and removes small decorative
  details while preserving the lizard-and-pen story.
- `concept-c-symbol.svg`: Reduces the idea to two strong arcs, an eye, and a pen
  tip. Intended to remain readable at small sizes.
- `concept-d-abstract.svg`: Treats the lizard and pen as a single continuous
  circular mark. This is the most abstract and scalable direction.

All concepts use a `1024 x 1024` view box and contain only editable SVG shapes.
