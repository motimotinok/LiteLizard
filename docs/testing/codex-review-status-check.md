# Codex review status check

This temporary document exists only to create a harmless pull request for
observing how GitHub displays the completion state of an automated Codex
review.

## Observation checklist

- The pull request is recognized as ready for review.
- The automated Codex review starts.
- GitHub updates the review or check state when the review finishes.
- Any review summary or inline comment is visible from the pull request.

This document is not intended to be merged into the product documentation.
