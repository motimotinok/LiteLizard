## 📋 実行タスク（優先度順）

### ルール（Codex 用）
- タスクは優先度の高い順に並べる（上にあるものほど優先）
- ダッシュボードの1タスクに集中する。完了したら次に進む
- タスク完了時:
  1. `docs/wbs.md` の該当タスクのステータスを `✅` に更新する
  2. このファイルの該当タスクを `✅ 完了済みタスク` セクションに移動する
  3. **ダッシュボードを次の最優先タスクに更新する**（必須）
  4. コミット & プッシュ → `dev` ベースで PR 作成
- 1タスク1ブランチ1PR。ブランチ名は `codex/{タスクID}`
- 詳細な仕様・完了条件は `docs/wbs.md` の「詳細」セクションを参照する
